# Termux Setup
Setup for termux
- Sets up zsh
- Installs ohmyzsh themes using installer from [https://github.com/cabbagec/termux-ohmyzsh](https://github.com/cabbagec/termux-ohmyzsh)
- Installs vim and git
- Installs fortune
- Adds arrow keys to keyboard.
